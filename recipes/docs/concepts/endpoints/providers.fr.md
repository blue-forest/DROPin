# Fournisseurs

Les Fournisseurs permettent de définir une communication avec un gestionnaire de paquets.

Seraient compatibles à terme :

- containers (Docker Hub, …)
- dépôts (GitHub, GitLab, …)
- npm (JavaScript)
- pip (Python)
- helm (Kubernetes)
- arch (pacman)
- nix
- apt (Debian)
- yum (Red Hat)
- gem (Ruby)
- maven (Java)
- gradle (Java)
- composer (PHP)
- conan (C/C++)
- nuget (C#)
- dotnet
