# Databases

Databases allow you to configure communications with an external database management system.

Would be compatible in the long run:

- MongoDB
- MySQL and MariaDB
- PostgreSQL
- Prometheus
- Redis
- Cassandra
- ElasticSearch
- RethinkDB
- InfluxDB
- SQLite
- Neo4j
- CouchDB
- ArangoDB
